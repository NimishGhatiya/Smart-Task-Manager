# Smart Task Manager

Full‑stack app with Next.js + Express, in‑memory data, task dependencies, and real‑time UI via fetch polling.

## Quick Start

### 1) Backend
```bash
cd server
npm i
npm run dev
# http://localhost:4000
```

### 2) Frontend
```bash
cd ../web
npm i
npm run dev
# http://localhost:3000
```

> Data is in memory. Restarting the server clears users/tasks.
